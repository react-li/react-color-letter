react-color-letter
===
彩色背景字母

![](./example/demo.png)


```js
git clone git@github.com:react-li/react-color-letter.git
npm install
npm start
```
在浏览器中打开 http://localhost:3031 即可查看例子


# 使用
`npm install react-color-letter --save`


```js
import ColorLetter from 'react-color-letter'


<ColorLetter 
  name="Julian"
  size={50}
  radius={20}
/>
```
